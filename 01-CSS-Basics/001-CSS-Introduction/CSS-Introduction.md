# CSS Introduction

## Introduction

CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation and appearance of HTML documents. It controls how web pages look by defining styles such as colors, fonts, spacing, layouts, animations, and responsive behavior.

CSS separates the presentation layer from the content layer, allowing developers to create visually appealing, consistent, and maintainable websites.

---

# Definition

CSS (Cascading Style Sheets) is a style sheet language that defines how HTML elements are displayed on web pages.

---

# What Does CSS Do?

CSS is responsible for:

- Styling web pages
- Applying colors
- Setting fonts
- Creating layouts
- Adding spacing
- Designing navigation bars
- Creating responsive websites
- Adding animations and transitions
- Improving user experience

# CSS Example

### HTML

```html
<h1>Welcome to CSS</h1>
<p>CSS makes web pages beautiful.</p>
```

### CSS

```css
h1{
    color: blue;
}

p{
    color: gray;
}
```

---

# Output

- Heading appears in blue.
- Paragraph appears in gray.

# Best Practices

- Keep HTML and CSS separate.
- Use external CSS whenever possible.
- Write meaningful class names.
- Maintain proper indentation.
- Avoid duplicate styles.
- Organize styles logically.

## 2. What does CSS stand for?

# Summary

CSS is one of the three core technologies of web development, alongside HTML and JavaScript.
