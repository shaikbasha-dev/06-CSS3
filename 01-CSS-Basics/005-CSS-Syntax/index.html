CSS Syntax

Every CSS rule consists of a selector and a declaration block. Understanding CSS syntax is the foundation for writing effective stylesheets.

Basic CSS Syntax


selector {

    property: value;

}

Components of CSS Syntax

  • Selector - Selects the HTML element to be styled.
  • Property - Specifies what style should be applied.
  • Value - Defines how the property should appear.
  • Declaration - A property-value pair.
  • Declaration Block - One or more declarations enclosed within curly braces.

Example


h1 {

    color: blue;

    font-size: 40px;

}

In the above example, h1 is the selector, color and font-size are properties, while blue and 40px are their corresponding values.

Important Syntax Rules

  1. Every declaration should end with a semicolon.
  2. Properties and values are separated by a colon.
  3. Declarations must be enclosed within curly braces.
  4. Write one declaration per line for better readability.
  5. Indent the code properly to improve maintainability.

A strong understanding of CSS syntax is essential before learning selectors, properties, layouts, animations, Flexbox, Grid, and responsive web design.
