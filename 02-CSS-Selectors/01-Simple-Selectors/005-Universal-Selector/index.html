CSS Universal Selector

The Universal Selector selects every HTML element on the webpage. It is represented by the asterisk (*) symbol and is commonly used to reset default browser styles.

Best Practices

  • Use the Universal Selector primarily for CSS resets.
  • Apply box-sizing: border-box globally.
  • Avoid unnecessary global styling.
  • Override global styles using specific selectors.
