Hyperlink Lifecycle

  1. Page loads → Link appears as :link
  2. Mouse pointer enters → :hover
  3. Mouse button pressed → :active
  4. Browser opens webpage
  5. Browser stores history
  6. Next visit → :visited

LVHA Rule


L → Link

V → Visited

H → Hover

A → Active

CSS should always define hyperlink pseudo-classes in this order.

Best Practices

  • Always follow the LVHA Rule.
  • Use meaningful hover effects.
  • Keep visited links distinguishable.
  • Provide clear visual feedback.
  • Maintain accessibility and sufficient color contrast.
