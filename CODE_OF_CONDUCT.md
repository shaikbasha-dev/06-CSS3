# Code of Conduct

## Our Commitment

This repository is committed to providing a welcoming, inclusive, respectful, and harassment-free environment for everyone, regardless of age, experience, gender identity, disability, ethnicity, nationality, religion, or background.

The goal is to foster a positive learning environment where knowledge can be shared openly and respectfully.

---

## Expected Behavior

All participants are expected to:

- Be respectful and professional.
- Use welcoming and inclusive language.
- Provide constructive feedback.
- Respect differing opinions and experiences.
- Focus on improving the educational quality of the repository.
- Help create a positive learning environment for everyone.

---

## Unacceptable Behavior

The following behaviors are not acceptable:

- Harassment or discrimination of any kind.
- Personal attacks or insulting comments.
- Offensive language or inappropriate content.
- Trolling, intimidation, or abusive behavior.
- Publishing private or confidential information without permission.
- Any conduct that negatively affects the learning community.

---

## Enforcement

Repository maintainers are responsible for clarifying and enforcing these standards.

Contributions, comments, or interactions that violate this Code of Conduct may be edited, removed, or rejected at the maintainers' discretion.

Repeated violations may result in temporary or permanent restrictions from participating in the project.

---

## Scope

This Code of Conduct applies to:

- Repository discussions
- Issues
- Pull Requests
- Code reviews
- Community interactions related to this repository

---

## Reporting

If you encounter behavior that violates this Code of Conduct, please open a private discussion with the repository maintainer through GitHub.

All reports will be reviewed fairly and handled respectfully.

---

## Acknowledgement

By participating in this project, you agree to follow and uphold the standards described in this Code of Conduct.

Together, we can build a professional, respectful, and valuable learning resource for the developer community.
